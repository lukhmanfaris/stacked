# Stacked

Your bookmarks, beautifully organized.

---

A bookmark manager that organizes saved links into visual collections and lets you share them as public pages. Built for people who actually use their bookmarks.

### Features

- **Stack View** — bookmarks displayed as layered card stacks, not flat lists
- **Smart Collections** — auto-categorize links by domain, title, or tag patterns
- **Portable Pages** — share curated bookmarks as a public start page
- **Command Palette** — CMD+K to navigate everything
- **Link Health** — automatic dead link detection
- **Import / Export** — bring bookmarks from any browser (HTML, JSON, CSV)
- **Instant Search** — full-text search with filters across all your links

### Built with

- [Next.js](https://nextjs.org) — React framework
- [Supabase](https://supabase.com) — Auth, database, storage
- [shadcn/ui](https://ui.shadcn.com) — Component primitives
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [TanStack Query](https://tanstack.com/query) — Data fetching
- [Framer Motion](https://motion.dev) — Animations
- TypeScript throughout

### Getting started

#### Prerequisites

- Node.js 18+
- A [Supabase](https://supabase.com) project (or use the local dev stack)

#### Setup

```bash
git clone https://github.com/lukhmanfaris/stacked.git
cd stacked
npm install
```

```bash
cp .env.example .env.local
# Fill in your Supabase credentials — see .env.example for details
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
